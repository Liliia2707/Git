1.Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?
  git checkout . (или один конкретный файл)
2.Как вернуть файл из индекса в рабочий каталог?
  git reset HEAD (и имя файла)
3.Как отменить ошибочный commit ?
  git reset --soft HEAD^
4.Как исправить коммит отправленный на сервер GitHub?
  git push --force
5.Как создать новую ветку и переключится в нее?
  git checkout -b (и имя новой ветки)
5.В чем отличение между fast-forward merge  and non fast-forfard merge?
  Fast-forward merge - это слияние веток без создания нового коммита, 
  а Non fast-forward merge - создание нового коммита обьединяет изменения разных веток и сохраняет их вместе.
6.Когда может возникнуть конфликт слияния веток, как  его разрешить?
  Например, если два человека сделали изменения в одной и той же строке.
  Нужно открыть файл,оставить только одни изменения,сохранить файл и создать коммит.
7.Можно ли удалить ветку? Как? 
  git branch -d (и имя ветки)
