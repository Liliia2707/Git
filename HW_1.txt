1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
-Открыть гит баш, указать путь к директории куда нужно скопировать
-Открыть гит хаб, нужный репозиторий и скопировать ссылку на него
-Написать команду git clone и вставить ссылку
2.Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
- на гит хабе нажать new repository, дать ему название,нажать create repository
- Открыть гит баш, указать путь к необходимому локальному каталогу
- git init
- git branch -M main
- скопировать ссылку в гит хабе с названием репозитория
- git remote add origin https://github.com/Liliia2707/primer.git
- git add . (если добавить все содержимое) или указать папки или файлы 
- git commit -m "Initial commit"
- git push origin main
3.Как получить на локальный компьютер изменения с github.com
- в гит баш проложить путь к директории,которая привязана к репозиторию
- git pull origin main
4.Какую информацию нам дают команда :
 git status
-на показывает,какие файлы были изменены,добавлены или удалены,показывает состояние дерева Git
 git log
- показывает историю коммитов в репозитории
 git remote -v 
- показывает список удаленных репозиториев,связанных с локальным репозиторием и URL-адреса, по которым можно получить или отправить изменения.
